# 로그인 및 게시판 웹 페이지

<br/>
<br/>

## 1. 개요

게시판 기능과 로그인 기능이 주된 웹 페이지 입니다.

<br/>
<br/>

## 2. 주요 기능

- 회원가입, 로그인
- 회원정보 수정, 삭제
- 게시글 작성, 수정, 삭제
- 답변 게시글 작성, 수정, 삭제



<br/>
<br/>

## 3. 사용 언어 및 도구

- JSP
- Javascript
- HTML/CSS
- jquery
- MYSQL
- eclipse/apache-tomcat
- autoset(MYSQL)
